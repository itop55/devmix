1 Универсальный селектор - обозначается звездочкой ( * ). Задает одновременно стили для всех элементов HTML страницы. Например, общий шрифт, цвет, размер текста, отступы и т.д.

CSS

1

2

3

4

5

6

* {
    font-family: "Roboto", sans-serif;
    color: blue;
    padding: 0;
    margin: 0;
}

</div>

2 Селектор элемента - указывает на определенный тег в вашем HTML-файле ( p, span, div, h2, ul ... ). Выбирает все элементы, соответствующие заданному имени тега.

Например, следующий код задаст для всех элементов <p> размер текста 20px и оранжевый цвет.

CSS

1

2

3

4

p {
    font-size: 20px;
    color: #ffa500;
}

3 Селектор класса - Выбирает все элементы, у которых есть данный класс. При стилизации элементов с помощью класса, в css перед названием класса необходимо ставить точку ( .big-text )

В HTML-файле класс пишем без точки <p class="big-text"> ... </p>

CSS

1

2

3

4

5

.big-text {
    text-decoration: underline;
    text-transform: uppercase;
    color: #b45cff;
}
HTML

1

2

3

4

5

6

<p class="big-text">Главные СМИ предупреждают</p>
<div>
    <span class="big-text">Цены на бензин начинают падать!</span>
    <p>Склады ломятся от зерна</p>
</div>
<p>Может показаться странным, но небо темнеет</p>

4 Селектор идентификатора (id) - id используется для стилизации одного, уникального элемента. Значение id не должно повторяться, в одном HTML-файле не может быть двух похожих id.

При стилизации элементов с помощью id, в css перед названием id необходимо ставить знак решетки ( #main-menu )

В HTML-файле id пишем без решетки <ul id="main-menu"> ... </ul>

CSS

1

2

3

4

5

#main-menu {
    list-style: square;
    font-weight: bold;
    color: #c74d00;
}

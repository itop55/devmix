HTML - это язык, который используется для создания сайтов. HTML отвечает за расположение в документе Ваших текстов, рисунков, таблиц. Иными словами, то что написано в HTML файле то и выводиться на экран.

Каждый HTML-файл представляет собой простой текстовый файл, который можно просматривать с помощью любого веб-браузера (например, Chrome, Yandex, Opera). Фактически, сайт который Вы сейчас видите, - это просто данные, которые были сохранены в файле HTML и отправлены в ваш браузер.

Обычно средний сайт включает несколько разных HTML-страниц. Например: главная страница, страница новостей, страница о компании, страница контактов и т.п.

Теги

HTML-теги - это ключевые слова, которые определяют, как ваш браузер должен выводить тот или иной контент(текст, картинки, видео). То есть в каком виде показывать содержимое HTML файла в браузере.

Теги нужно писать в угловых скобках: <название тега>

Большинство тегов состоят из двух частей:
открывающей - <...>
закрывающей - </...>

Например:
<video> - это открывающий тег
</video> - это закрывающий тег

Обратите внимание, что закрывающий тег отличается от открывающего лишь тем, что имеет дополнительный символ косой черты - /

Необходимо раз и навсегда запомнить что если есть открывающий тег <...> то обязательно должен быть и закрывающий </...>

Вложенные теги и порядок написания тегов

Большинство тегов могут содержать внутри себя сколько угодно вложеных тегов. Представим что тег - это ящик в который мы можем положить ящик поменьше, в свою очередь в этот маленький ящик мы можем положить ящик еще меньше и так дальше.

Каждый открытый тег должен обязательно закрываться, как и с ящиками у которых есть крышки, если у нас есть три яшика вложенных в друг друга, то мы сначала должны закрыть самый маленький ящик, потом ящик сверху него, потом самый верхний ящик.

   </Тег "ящик средний">

Если тег открывается внутри другого, например тег <p> открывается внутри тега <div>, то тег <p> должен закрыться до закрытия внешнего тега <div>.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Мой первый сайт</title>
  </head>
  <body>
    Содержимое моего сайта
  </body>
</html>

  1. <!DOCTYPE html> - (доктайп) указывает тип документа. Так мы говорим браузеру, что это html страница

  2. <html>...</html> - открывающий тег <html> задает начало страницы, а закрывающий </html> соответственно конец. Все остальное (теги, текст, картинки, видео и т.д) помещаеться между этим тегом. В документе должен быть только один тег <html>...</html>

  3. <head>...</head> - тег для размещения служебной информации для браузера. Здесь задаем название сайта, указываем кодировку страницы, при необходимости подключаем файл стилей

  4. <meta charset="UTF-8"> - тег который сообщает браузеру, что символы в данном документе закодированы в соответствии с кодировкой UTF-8

  5. <title>Мой первый сайт</title> - здесь указваем название сайта, которое будет выводиться во вкладке браузера, как показано ниже:

  6. <body>...</body> - тег, в котором размещается все содержимое сайта(текст, картинки, видео, таблицы, списки и т.д.) Все, что находится между тегами <body> и </body> будет выводиться браузером на экран.

Выше мы перечислили шесть основных тегов HTML, их нельзя дублировать и они должны быть в правильном порядке, как показано в примере.

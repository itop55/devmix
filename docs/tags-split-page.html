Теги для разбивки HTML страницы

HTML
  • <p>

Обычно HTML страница состоит из четырех основных частей:

  1. <header>...</header>

    Header - Верхняя часть (шапка сайта). Содержит контент, относящийся ко всем страницам сайта, например меню, логотип, название сайта.

  <h2>Мой первый сайт</h2>
  <p>Учебный портал - HTML/CSS</p>
</header>

    Мой первый сайт

    Учебный портал - HTML/CSS

  2. <nav>...</nav>

    Sidebar - Боковое меню. Может быть размещен слева или справа от основного контента. Для создания бокового меню используйте тег <nav>...</nav>

    <nav>...</nav> - Используется для создания меню. На странице может быть несколько элементов <nav>, например, один для бокового меню в sidebar, а второй для основного меню в header.

  3. <main>...</main>

    Main - Основная часть. Занимает самую большую часть HTML-страницы. Содержит контент, который относится только к данной странице. main в свою очередь делиться на отдельные блоки с помощью тега <section>...</section>

    <section>...</section> - используется для создания отдельных блоков в документе, например, блок новостей, галереи, товаров и т.д.

  4. <footer>...</footer>

    Footer - Нижняя часть сайта(подвал сайта). Содержит контактную информацию, график работы, ссылки на социальные сети, юридическую информацию. Как и шапка сайта, footer так же является общим блоком для всех страниц сайта.

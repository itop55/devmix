Урок 3

HTML теги для работы с текстом

В этом уроке 3

HTML
  • <p>
  • <span>
  • <strong>
  • <br>
  • <h1> - <h6>

<p>...</p>

<p> используется для добавления текста на сайт. Текст добавленный с помощью тега <p> всегда выводиться с новой строки.
<p> по умолчанию занимает всю доступную ему ширину, а высота будет зависеть от количества текста.

1

<p>Какой-то текст</p>

<span>...</span>

<span> используется для стилизации частей текста. К примеру, если в тексте необходимо выделить одно слово красным цветом, то нужно обернуть его в тег <span> и для него писать css стили.

1

<p>Чистосердечное <span class="text-red">признание</span> облегчает душу</p>

<strong>...</strong>

<strong> используеться для выделения текста жирным шрифтом.

1

<p>Оказывается, обучение кадров - <strong>приоритетная задача</strong></p>

Оказывается, обучение кадров - приоритетная задача

<br>

<br> переносит текст на следующую строку.

1

<p>Оказывается, <br>цены на бензин начинают падать, <br>жизнь прекрасна</p>

Оказывается,
цены на бензин начинают падать,
жизнь прекрасна

<h1>...</h1>

Для определения заголовка используются теги <h1> - <h6>
Всего существует шесть тегов заголовков: <h1> <h2> <h3> <h4> <h5> <h6>

<h1> наиболее важный, должен использоваться для основного заголовка страницы, а затем заголовок <h2>, менее важные <h3>, и так далее.

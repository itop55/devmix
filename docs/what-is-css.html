Что такое CSS?

В этом уроке 1

CSS
  • CSS селектор
  • Блок css правил

Объявление CSS стилей состоит из двух частей:
1. Селектор - элемент html страницы, который необходимо стилизовать.
2. Блок css правил - список правил для данного селектора

Каждое css правило состоит из свойства и значения, разделенных двоеточием : и оканчивающихся точкой с запятой ;, а сами правила группируются в фигурные скобки { }

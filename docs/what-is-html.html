Что такое HTML?

HTML - это язык, который используется для создания веб-страниц. HTML буквально предназначен для того, чтобы сообщить вашему браузеру: «Вот то что нужно показать на экране. Сделай это." То есть, что написано в HTML файле то и выводиться на экран.

Каждый HTML-файл представляет собой простой текстовый файл, который можно просматривать с помощью любого веб-браузера (например, Chrome, Yandex, Opera или Safari). Фактически, вся информация, которую вы сейчас читаете, - это просто данные, которые были сохранены в файле HTML и отправлены в ваш браузер.

Обычно средний веб-сайт включает несколько разных HTML-страниц. Например: главная страница, страница новостей, страница о компании, страница контактов и т.п.

Теги

HTML-теги - это ключевые слова, которые определяют, как ваш браузер должен выводить тот или иной контент. То есть в каком виде показывать содержимое HTML файла в браузере.

Большинство тегов состоят из двух частей:
открывающей - <...>
закрывающей - </...>.

Например:
<html> - это открывающий тег
</html> - это закрывающий тег

Обратите внимание, что закрывающий тег отличается от открывающего лишь тем, что имеет дополнительный символ косой черты - /
Каждый открытый тег должен обязательно закрываться

Также есть теги, которые имеют только открывающий тег (закрывающий тег не требуется)

Например:
<br> - задает перенос строки
<hr> - задает горизонтальную линию
